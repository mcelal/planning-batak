## Installation

Install Front-End

```sh
yarn install
```

Install SocketIO Server

```sh
cd /server
yarn install
```

Change .env files
```
cp .env-example .env
cd /server
cp .env-example .env
```

## Usage

Start Vue project

```sh
yarn run serve
```

Start SockerIO server
```
cd /server
node server.js
```
